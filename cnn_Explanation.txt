Explanation of Approach

In this project, a Convolutional Neural Network (CNN) was built from scratch using the CIFAR-10 dataset for multi-class image classification. The dataset contains 10 different image categories, and the images were normalized by dividing the pixel values by 255 to make the training process more stable and efficient. The class labels were converted into categorical format using one-hot encoding to ensure proper tensor shape handling during model training.

The CNN architecture was created using convolution layers to extract image features, batch normalization layers to improve training stability, max pooling layers to reduce image dimensions, and dropout layers to reduce overfitting. After feature extraction, flatten and dense layers were used for classification. ReLU activation function was applied in the hidden layers, while Softmax activation was used in the output layer to classify the images into 10 classes.

The model was trained using the Adam optimizer and categorical crossentropy loss function. Early stopping and dropout techniques were used to improve generalization performance and prevent overfitting. Finally, the model was evaluated on the test dataset and achieved good classification accuracy while keeping the architecture simple and efficient without using any pretrained models.
